== Description ==

Personaliza os campos do APPMAX no Checkout Mestres do WP.

== Installation ==

= Instalação automática pelo Wordpress =

1. Selecione **Add New** em seu WordPress na aba **Plugins**.
2. Procure por **Checkout Mestres do WP Addon AppMax**.
3. Clique em **Install Now** e **Activate Plugin**.

= For manual installation via FTP: =

1. Faça o upload do **checkout-mwp-addon-appmax** na pasta **/wp-content/plugins/**.
2. Ative na seção **Plugins** do seu Wordpress.
